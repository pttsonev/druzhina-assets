02Execution flow — video to bid

The Frank spine as it runs today. Amber nodes wait for you; everything between them runs autonomously. Artifact chips show exactly what lands on disk at each step.

Detect cuts vidin · stage 1 · no LLM

ffmpeg scene scores → exact frame ranges assembled in Python. VFR fails loudly. Run dir is pinned to the video by sha256 and to the show.

python3 -m vidin.run edit.mp4 --show FRANK --run-dir runs/frank-v001-t02 --threshold 0.2 --stages 1
manifest.jsonshots.jsonthumbs/*.jpgcontact_sheet_*.jpgmapping.yaml (template)

Map segments → shot codes you · the hard gate

Contact sheets open, fill code: per segment. Same code twice = split shot (merges). Invisible fixes → expected_misses. Never overwritten by any rerun.

mapping.yaml (your edits)

Triage every segment vidin · stage 2 · small VLM

Two 1024px frames per segment → grammar-locked {vfx_likely: bool}. The show's client brief (brick wall) is injected as wrapped data. One file per model; resumable mid-run; 5xx retried.

python3 -m vidin.run ... --stages 2 --resume # qwen3.5:9b python3 -m vidin.run ... --stages 2 --model X --resume # any other

Evaluate vs the bid seed vidin · pure python

Detection match rate (36 / 34-in-v5), per-model precision · recall · f1, named false negatives, brief-attributable hits. Rerun freely — read-and-compute only.

Approve the brain write you · db gate

Dry run prints every row first; --confirm writes frame ranges keyed by shot code, correlation-tagged, ambiguity refused with rollback.

LLM proposes preset + seniority per shot (closed vocab, off-vocab → needs_human); then price_show() runs the two-axis math. The v5 toggles reproduce your real bid to the dollar: 23,284 → 29,105 → 41,747 → 37,573.

Diff blind bid vs your real bid botev · P0 drz-5rj

The point of the spine: Vidin segments → Botev prices, with nobody peeking at the answer — then line-by-line diff against the canonical export. The gap is the studio's calibration number.

↺ Rebid loop (new edit lands): fresh run dir → stages 1–2 → quick re-map → python3 -m vidin.edit_diff --old runs/frank-v001-t02 --new runs/frank-v002 → per-code longer / shorter / added / removed → Botev re-prices only the worklist.
